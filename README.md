
# Fix Readme MyRecipe

This repository hosts the MyRecipe application, designed to help you organize and manage your favorite recipes efficiently. The app integrates with the Gemini API to enhance its functionality.

## Setup Instructions

Follow these steps to get started with the project:

1. **Install Dependencies**:
   Run the following command to install the necessary packages:
   ```bash
   npm install

2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
